**Audience:** Anyone navigating the source tree — developers, reviewers, or readers seeking the Agda module behind a specific theorem or concept.

## How to Read This Index

| Column | Meaning |
|--------|---------|
| **Module** | Agda module path (relative to `src/`) |
| **Description** | One-line summary of what the module defines or proves |
| **Gen** | `—` = hand-written; `PY` = auto-generated by Python oracle; `DEAD` = superseded, not on critical path |
| **Doc** | Primary documentation cross-reference |

Modules are grouped by directory (layer). Within each layer, modules are listed in approximate dependency order — earlier modules are imported by later ones.
